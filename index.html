goshortener by luckydev

Ruby gem/library to use Google URL Shortener (goo.gl) service and shorten long URLs

Use GoShortener to shorten Long URLs to http://goo.gl format short URLs. Drop in the gem to your project, and start using it as given in README file.

Dependencies

json, rest-client

Install

gem install goshortener
